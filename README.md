# Description
This is the RPG made by Unity, it has been applied Agora Engine, MySQL, Photon Network.
# Feature 
- Player movement
- Earn money
- Pick up items and store in inventory
- Kill enemies
- Buy things at the marketplace
- Video calls
- Multi-player mode
